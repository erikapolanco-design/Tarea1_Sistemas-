# 🧠 INFORMACIÓN SOBRE GITHUB

- Conocido como la red social de los desarrolladores
- GitHub utiliza Git, un sistema de control de versiones

  
GITHUB es una plataforma online que aloja de forma correcta:
- Codigos (repositorios) en la nube
- Trabajar en esquipo dentro del proyecto de forma simultánea
- Permite volver a versiones anteriores si ocurre un error
- Marcar proyectos, seguir a perfiles y contribuir a proyectos abiertos
- Automatiza flujos de trabajo
- Subir archivos o códigos

### Principalmente:
¡Guardar, compartir y gestionar proyectos de programación!

# 🔧 ¿CÓMO FUNCIONA?
## De lo más básico:

Los títulos -> #
 - Sirven para hacer letras grandes

Separar -> ---
 - Sirven para dividir partes o diapositivas

Texto normal
 - Escribir normal, sin simbolos

Listas de cosas -> -
 - Para enumerar

Descargar proyecto -> ZIP
  - Para guardar información en formato ZIP
    PASOS:
     - Entra al repositorio
     - Haz clic en el botón “Code”
     - Presiona “Download ZIP”
     - Se descarga el archivo
     - Lo descomprimes en tu computador

Descarga proyecto -> GIT
  - Para guardar información en formato GIT
    PASOS: 
     - Copias el link del repositorio
     - En tu computador escribes: "git clone link-del-repositorio"
